# 60 — Presión de decisión

## Propósito

Validar que verdad financiera se traduzca en criterio útil sin sobreprometer consejo.

## Alcance

- liquidez inmediata
- presión futura
- fragilidad operativa
- capacidad real de decidir

## Tipos de escenarios

- puedo gastar o no
- puedo mover valor o no
- conviene esperar o actuar
- tensión próxima invisible al saldo

## Casos límite

- mismo saldo con compromisos muy distintos
- ingreso probable usado como base de consumo
- compra que mejora cobertura pero rompe caja

## Criterios de aceptación

- recomendación nace de estado calculado
- no hay consejo fuera de evidencia
- tradeoff principal queda explícito
