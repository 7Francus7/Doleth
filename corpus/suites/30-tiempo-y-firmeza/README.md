# 30 — Tiempo y firmeza

## Propósito

Validar que Doleth preserve cuándo impacta valor y qué tan confiable es.

## Alcance

- tiempo efectivo
- tiempo de caja
- firmeza de relaciones
- degradación o mejora de certeza

## Tipos de escenarios

- cuotas
- cobros diferidos
- compromisos futuros
- ingresos probables

## Casos límite

- preautorizaciones
- cheque diferido
- pago confirmado pero no acreditado
- derecho de cobro deteriorado

## Criterios de aceptación

- tiempo y firmeza quedan explícitos
- verdad provisional no se maquilla como consolidada
- lectura de decisión cambia con horizonte y certeza
