# 90 — Casos que rompen

## Propósito

Mantener frontera explícita del modelo y obligar a Doleth a reconocer límites.

## Alcance

- casos patológicos
- contradicciones
- realidades subdeterminadas
- fenómenos todavía no modelables

## Tipos de escenarios

- conflicto ontológico
- conflicto de estados
- evidencia incompatible
- decisión imposible

## Casos límite

- misma realidad exige dos interpretaciones incompatibles
- falta dato imposible de obtener
- institución local crea soporte no absorbido por modelo actual

## Criterios de aceptación

- sistema falla de forma explícita
- ruptura queda tipificada
- caso alimenta roadmap conceptual, no improvisación silenciosa
