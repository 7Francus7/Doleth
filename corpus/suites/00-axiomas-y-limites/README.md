# 00 — Axiomas y límites

## Propósito

Probar verdades fundacionales de Doleth y detectar interpretaciones que jamás deben pasar.

## Alcance

- invariantes del modelo
- confusiones primitivas del dominio
- fronteras entre fenómeno real y lectura superficial

## Tipos de escenarios

- transferencia interna que parece ingreso
- pago que parece gasto pero cancela obligación
- compra de activo que parece consumo
- saldo alto con liquidez baja

## Casos límite

- misma evidencia soporta lectura intuitiva errónea
- usuario usa lenguaje cotidiano que empuja clasificación superficial

## Criterios de aceptación

- caso no rompe invariantes globales
- sistema rechaza lectura prohibida
- explicación final sigue siendo humana y trazable
