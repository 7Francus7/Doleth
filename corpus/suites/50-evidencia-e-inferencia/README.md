# 50 — Evidencia e inferencia

## Propósito

Validar cómo Doleth convierte señales imperfectas en hipótesis, verdad provisional y verdad consolidada.

## Alcance

- señales textuales
- señales estructuradas
- fuentes múltiples
- conflictos de evidencia

## Tipos de escenarios

- mensaje claro
- mensaje ambiguo
- fuentes que confirman
- fuentes que chocan

## Casos límite

- texto sin monto
- PDF incompleto
- una entrada contiene varios hechos
- dos fuentes parecen mismo hecho pero discrepan

## Criterios de aceptación

- inferencia es trazable
- sistema pregunta cuando costo de asumir es alto
- misma realidad converge al mismo modelo final cuando evidencia alcanza
