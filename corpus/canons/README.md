# Canons

Canons son subconjuntos congelados del corpus.

No maximizan cobertura total.
Maximizan estabilidad conceptual.

Reglas:

- todo canon define gate permanente
- todo caso dentro de canon debe ser nombrable
- todo caso dentro de canon debe ser `regression_locked`
- cambio en canon exige explicación de alto nivel

Canon inicial:

- `canon-1.0`
