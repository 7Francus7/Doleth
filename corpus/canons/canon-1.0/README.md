# Canon 1.0

Canon 1.0 congela primeros 50 escenarios fundacionales de Doleth.

Objetivo:
definir base mínima que toda versión futura deberá comprender correctamente antes de reclamar evolución conceptual.

## Reglas

- todos los casos deben tener nombre semántico
- todos los casos deben mapear a suites
- todos los casos deben terminar en `regression_locked`
- grupo E puede terminar en failure mode explícito, no necesariamente en “resolución limpia”
